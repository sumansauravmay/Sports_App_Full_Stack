## Run Locally

Clone the project

```bash
  git clone https://github.com/sumansauravmay/Sports_App_Full_Stack.git
```

Go to the project directory

```bash
  cd 
Sports_App_Full_Stack

```

Install dependencies for frontend as well as backend

```bash
  cd frontend
  npm install
```

```bash
  cd backend
  npm install
```

Start the localhost server

```bash
  cd frontend
  npm start
```
